# Sarponesia Backend

## Deskripsi
Aplikasi untuk mengelola backend layanan Sarponesia (frontend dikerjakan terpisah oleh rekan tim, namun tetap dapat diakses di repo ini).

## Tech Stack
- Backend: Laravel
- Frontend: Blade / React / Vue
- Database: MySQL

## Fitur
- Login & Role
- CRUD Data
- Relasi Tabel

## Cara Menjalankan
1. `git clone <repo-url>`
2. `composer install`
3. `php artisan migrate`
4. `php artisan serve`
